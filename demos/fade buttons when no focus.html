Sudo Slider jQuery Plugin - Hide nav when not on slider

Usage

This is everything you need.

Include the javascripts

<script type="Text/Javascript" src="http://code.jquery.com/jquery-1.11.2.min.js"></script>
<script type="text/javascript" src="../js/jquery.sudoSlider.min.js"></script>

The Javascript to start it. (This script has to be below your CSS)

<script type="text/javascript" >
    $(document).ready(function(){
        var sudoSlider = $("#slider").sudoSlider({
            controlsFade: false	
        });
        $(".prevBtn, .nextBtn, #slider").hover(
            function () {
                // Mousein
                $(".prevBtn, .nextBtn").stop().fadeTo(200, 1);
            }, 
            function () {
                //Mouse out
                $(".prevBtn, .nextBtn").stop().fadeTo(200, 0);
            }
        );
        $(".prevBtn, .nextBtn").stop().fadeTo(0, 0);
    });
</script>
