Help

In the top of the page is the slider, this shows how the slider behaves when the options selected below has been chosen.
You can change the options and the content below, and the slider will immediately reflect the change.
You can also open a popup with the slider instead, by pressing the "Open slider in popup" button on the right, the slider will still update immediately.

You can get HTML to use by pressing the "Get the result" button on the right, in the popup that opens you can also see the options to insert, or you can export/import the configuration you have the tool in.

The buttons under the slider controls the slider you see, and all also the slider in the popup (if you have one open).

To get started, try to find the "continuous" option and enable it (try to put "conti" in the "Filter options by name" input on the left).
Then press the "Get the result" button on the right, to see how the HTML that initializes the slider in this configuration looks.

If you want to know what the different options do, check out the documentation.
And check out the demos to see examples of what you can do. Including things that aren't currently possible to do using this tool.
